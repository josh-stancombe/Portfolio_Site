Rock, Paper, Scissors Games

Python 3

Project Specification

  • Using Python, recreate the game 'Rock, Paper, Scissors' for a user to play against the computer.
  • The first player to win 3 rounds wins the game.
  • Your application should be able to handle incorrect user input.
  • Your application should randomly generate the computers moves.
